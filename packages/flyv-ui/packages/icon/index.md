## Icon 图标

### 使用指南
``` javascript
import { Icon } from 'flyv';

Vue.use(Icon);
```

### 代码演示

#### 基础用法

```html
<flyv-icon :name="'media-pause'"/>
```

### API
| 参数 | 说明 | 类型 | 默认值 | 版本 |
|------|------|------|------|------|
| name | 图标名称 | `String` | - | - |

### Event

| 事件名 | 说明 | 参数 |
|------|------|------|
| click | 点击图标时触发 | - |
